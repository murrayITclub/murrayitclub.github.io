Favicons

A favicon is the tiny picture that appears in the tab. For example, this page's favicon is a blue laptop.

To make your own favicon...

1. First, open Piskel. Click Create Sprite, and then use the drawing tools to create a small picture.

2. Download your picture: click Export, then PNG, then click the bottom Download button.

3. Go to favicon-generator.org. Upload the picture you just downloaded from Piskel, choose Generate only 16x16 favicon.ico, and click Create Favicon.

4. Next, you'll see a screen like this. Click the link that says Download the generated favicon, NOT any other buttons (they're an ad).

5. That should make a file named favicon.ico save into your computer's Downloads folder. Move that file into your Everything IT folder, and then upload it to Github. If you need help with this last step, Jaxson is a great classmate to ask!

Note: you probably won't see your favicon right away - Github can take up to an hour to make this change. Don't panic - if you have a file called favicon.ico on Github, it will work eventually. Be patient!
