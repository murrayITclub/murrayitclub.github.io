Lua: A Custom Hour of Code

Lua is a fantastic scripting language common in a number of game applications (including, of course, Roblox). While it's useful and no harder to learn than any of its relatives, there don't seem to be quick'n'easy tutorials for getting started. Here's how you can dip your toes in the Lua pool:

First, you'll need an online space where you can write and run Lua programs. Here's how to set that up:

1. Go to repl.it, and create an account.

2. Click the + new repl button, and search for Lua as the language. Choose a name for your repl (it can be whatever you want!), and don't worry about the other options. Click Create repl.

3. Now you're ready to start writing Lua! You can write code in the white space in the middle, then click the run > button to see the results in the black box on the right. For example...

...hello world is always a good place to start.

4. Ready to learn more? Here's a solid video introduction (note: video starts partway in because the first 7 minutes describe installing Lua on your computer. You don't need to do that - just use your repl instead!)
