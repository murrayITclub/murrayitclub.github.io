How do I add colors, fonts, or other styles to my website?

Right now, your website has content (words and images) and structure (HTML tags). If you want to add styles, like colors and fonts, you'll need to use a new language called CSS.

1. First, you can use some code.org activities to learn how CSS works.

In Lesson 10: Try activities 1-12. (Yes, that's a lot, but you're learning a whole new language!)

Let's go!

Make sure you read all the directions for each activity - you will usually need to scroll down.

2. To create a stylesheet in Brackets, click File, then New.

Before you start typing, save the file: click File, then Save. Name the file style.css, and make sure to save it in your IT Club folder. This is very important, so ask for help if you need it!

Next, go back to your index.html file, and make sure there's a link element inside the head, like this:

Note: do not add all of the tags in that picture - just see if the highlighted line is already between <head> and </head>. If it's not there, add it!

Now you're ready to start adding styles to your website! Remember, every time you make a change to index.html or style.css, you'll need to save the file you changed, then reload your website to see the changes.
